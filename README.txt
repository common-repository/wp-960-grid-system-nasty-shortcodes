== Description ==

<p>
960 Grid System is an effort to streamline web development workflow by providing commonly used dimensions, based on a width of 960 pixels and this plugin helps you format your wordpress posts and pages easily by providing shortcodes. Your WordPress theme must be 960gs ready for this plugin to work efficiently.
</p>
<p>
<a href="http://opensource.clubkoncepto.com/wordpress-960-grid-system-shortcodes">Documentation</a>
</p>

== Installation ==


1. Upload this plugin to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. For complete reference on how to use this plugi, visit our <a href="http://opensource.clubkoncepto.com/wordpress-960-grid-system-shortcodes">documentation.</a>
